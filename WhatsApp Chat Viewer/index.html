v0.2

1
Préparez l'archive

Sur votre téléphone, ouvrez WhatsApp → ouvrez la conversation → appuyez sur ⋮ (menu) → « Exporter la discussion » → choisissez « Joindre les médias » (ou sans médias pour un fichier plus léger). Cela crée un fichier .zip.

2
Transférez sur votre ordinateur

Envoyez le fichier .zip par e-mail, AirDrop, Google Drive, câble USB ou tout autre moyen de transfert.

3
Visualisez dans l'application

Glissez le fichier .zip sur cette page (ou cliquez sur « Choisir des fichiers »). Sélectionnez votre nom dans la liste, et votre conversation s'affiche exactement comme sur votre téléphone.
